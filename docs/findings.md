# 서울시 공공자전거 수요 예측 분석 보고서

## 1. 데이터 구축

2023년 1월부터 2024년 12월까지의 원천 이용 88,708,877건을 GPS 기반 3개 군집과 1시간 단위로 집계했습니다. 최종 모델링 데이터는 17,544개 시간 × 3개 군집, 52,632행입니다.

## 2. 공간 군집화

대여소 좌표를 표준화한 뒤 K-Means를 적용했습니다. Inertia와 Silhouette score만으로 군집 수를 정하지 않고, 군집별 대여소 수·지리적 분리·해석 가능성을 함께 검토해 `K=3`을 선정했습니다.

## 3. 시계열 검증

무작위 분할을 사용하지 않고 학습 18개월, 검증 3개월, 테스트 3개월로 시간 순서를 보존했습니다. 시차변수는 예측 시점 이전의 실측치로만 생성했고, 테스트 확인 전에 모델 설정을 고정했습니다.

## 4. 모델 성능

| 모델 | Macro RMSE | Macro MAE |
|---|---:|---:|
| LightGBM | 190.724 | 113.054 |
| SARIMA | 460.629 | 260.015 |
| Hourly Naive | 725.571 | 456.731 |
| Weekly Naive | 776.525 | 427.048 |
| Daily Naive | 865.435 | 464.011 |

LightGBM은 SARIMA 대비 RMSE를 58.595%, Hourly Naive 대비 73.714% 낮춰 단기 연속성과 일·주 반복 패턴을 결합한 효과를 보였습니다.

## 5. 해석 주의사항

- 예측 단위는 개별 대여소가 아닌 GPS 기반 3개 공간 군집입니다.
- 절대 오차는 수요 규모가 큰 군집에서 더 크게 나타날 수 있습니다.
- 미래 실측 기상값은 예측 시점에 알 수 없어 입력에서 제외했습니다.
- 변수 중요도는 예측 기여도이며 인과효과가 아닙니다.
- 대표 그래프의 마지막 1주 표시와 전체 3개월 테스트 지표를 구분해야 합니다.

세부 산출물과 파일별 의미는 [`../outputs/README.md`](../outputs/README.md)에서 확인할 수 있습니다.
