# 포트폴리오 프로젝트 설명

## 프로젝트명

서울시 공공자전거 GPS 군집 기반 수요 예측 — 시간 누출을 방지한 군집별 1시간 수요 모델링

## 한 줄 소개

88,708,877건의 공공자전거 이용정보를 GPS 기반 3개 공간 군집으로 집계하고, LightGBM으로 SARIMA 대비 Macro RMSE를 58.595% 낮춘 다음 1시간 수요 예측 프로젝트입니다.

## 수행 내용

- 대여소 ID 표준화와 GPS 결측·0·서울 범위 밖 좌표 검증
- K-Means 후보의 Inertia·Silhouette·공간 분포를 비교해 `K=3` 선정
- 17,544시간 × 3개 군집의 수요 시계열 52,632행 구축
- 미래 정보 누출을 방지한 시간 순서 기반 학습·검증·테스트 분할
- Hourly·Daily·Weekly Naive, SARIMA, LightGBM의 Macro RMSE·MAE 비교
- 미확인 이상 구간을 임의 수정하지 않고 품질 감사 결과로 별도 보존

## 핵심 결과

- LightGBM Macro RMSE 190.724, Macro MAE 113.054
- SARIMA 대비 RMSE 58.595%, MAE 56.520% 개선
- Hourly Naive 대비 RMSE 73.714% 개선
- `lag_1`, `lag_168`, `lag_24`가 주요 예측 정보로 활용됨

## 사용 기술

Python, pandas, NumPy, scikit-learn, statsmodels, LightGBM, Matplotlib, JupyterLab, Git

## 강조 역량

대용량 공공데이터 검수, 공간 군집화, 시계열 누출 방지, 기준모델 설계, 예측 모델 검증, 이상 구간 기록, 재현 가능한 분석 문서화

## 해석 원칙

성능 수치의 비교 조건을 명확히 고정합니다. 현재 결과는 개별 대여소가 아닌 3개 공간 군집의 총수요 예측이며, 2024년 4분기 테스트에 한정됩니다. Gain 기반 중요도는 예측 기여도로만 해석하고 인과효과로 해석하지 않습니다. 재배치 활용 가능성은 제시하되, 재고·거치대·이동 비용을 포함한 운영 검증이 별도로 필요함을 밝힙니다.
